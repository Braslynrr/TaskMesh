# Frontend

**Local URL:** `http://localhost:3000`

## Requirements

- Node.js >= 20  
- Backend running at `http://localhost:5000`

## Configuration

Create a `.env` file with the following variable:

```env
NEXT_PUBLIC_API_URL=http://localhost:5000/api
```

Install dependencies:

npm ci

Start development server:

npm run dev

Build and start:

npm run build
npm start

Tests:

npm test
